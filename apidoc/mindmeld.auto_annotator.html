class mindmeld.auto_annotator.SpacyAnnotator(app_path, config=None)[source]¶

Bases: mindmeld.auto_annotator.Annotator

(English) Annotator class that uses spacy to generate annotations. Supported entities include: "sys_time", "sys_interval", "sys_duration", "sys_number", "sys_amount-of-money", "sys_distance", "sys_weight", "sys_ordinal", "sys_quantity", "sys_percent", "sys_org", "sys_loc", "sys_person", "sys_gpe", "sys_norp", "sys_fac", "sys_product", "sys_event", "sys_law", "sys_langauge", "sys_work-of-art", "sys_other-quantity", For more information on the supported entities for the Spacy Annotator check the MindMeld docs.
